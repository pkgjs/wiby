## wiby test

Test your dependents

```
Options:
  --version    Show version number                                     [boolean]
  --help       Show help                                               [boolean]
  --dependent  URL of a dependent                            [string] [required]
```

## wiby result

Fetch the results of your tests

```
Options:
  --version    Show version number                                     [boolean]
  --help       Show help                                               [boolean]
  --dependent  URL of a dependent                            [string] [required]
```
